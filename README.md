# Pandas Study – Titanic & Shopping Data Analysis

## Project Overview
This project was created to practice Python data analysis using numpy and pandas.

The project includes:
- Titanic dataset exploration and analysis
- Group-based statistics using pandas
- Missing value handling
- Shopping mall sales data preprocessing
- Git/GitHub workflow practice
